# ConversorDeDivisas
Conversor de moneda Alura one oracle
Con esta programa podremos convertir algunas divisas de diferentes paises como Estados Unidos, Colombia, Brasil, Chile entre otros. 
Una manera fácil y rápida de sacar tus cuentas al momento de viajar.
La tasa esta siempre actualizada con los ultimos indicadores que muestra el mercado gracias a una API.
